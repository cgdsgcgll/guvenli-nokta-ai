# Sprint Review

## Sprint 1'de Tamamlananlar

- GitHub repository oluşturuldu.
- Takım üyeleri ve rolleri belirlendi.
- Ürün fikri Güvenli Nokta AI olarak netleştirildi.
- README dosyası proje bilgilerini içerecek şekilde düzenlendi.
- Product backlog oluşturuldu.
- GitHub Projects üzerinde sprint board hazırlandı.
- Daily Scrum notları eklendi.
- Ana sayfa, ev bilgileri formu ve risk analizi sonuç ekranı için wireframe tasarımları hazırlandı.

## Ürün Durumu

Sprint 1 sonunda ürünün temel fikri, hedef kitlesi, problem tanımı, çözüm yaklaşımı ve temel özellikleri belirlenmiştir. Ürün henüz kod geliştirme aşamasında değildir; ancak geliştirme süreci için gerekli planlama ve görsel taslaklar hazırlanmıştır.

## Gösterilen Çıktılar

- Product backlog
- Sprint board
- Ana sayfa taslağı
- Ev bilgileri formu taslağı
- Risk analizi sonuç ekranı taslağı
